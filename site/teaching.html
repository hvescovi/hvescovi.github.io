  • Temas para o desenvolvimento de trabalho de conclusão de curso em Ciência da Computação:
    1. (18/12/2024) Sistema de páginas web que gera conteúdo estático, similar à plataforma Lattes. Um cadastro dinâmico mas que atualiza páginas estáticas. Referências:
      • https://ieeexplore.ieee.org/abstract/document/8974709
      • https://core.ac.uk/download/pdf/83597655.pdf
    2. (24/11/2023) T1) Sistema de cadastro (CRUD) tolerante a faltas, visando popularizar esse tipo de recurso caro e necessário. A ideia principal é permitir que um frontend acesse mais de um backend. É preciso também que o frontend esteja disponível em mais de uma fonte. A prova de conceito por meio de um CRUD torna possível a expansão da arquitetura para outras necessidades.
    3. (18/10/2023) T2) Sistema tolerante a faltas conforme demanda. Pode-se ligar ou desligar a tolerância a faltas. Isso é essencial pois sistemas tolerantes a faltas são caros.
    4. (24/11/2023) T3) Acréscimo de recursos no sistema aberto de perguntas (Serena). O ideal é reconstruir o sistema usando tecnologias mais modernas, fazer um bom projeto do sistema e tornar o sistema mais acessível (hospedado na internet ou no computador do professor).
    5. (23/02/2024) T4) Desenvolvimento de sistema web (ex: Serena) para hardware embarcado (ex: raspberry).
  • Para usar o git, você precisa de apenas 7 comandos do terminal, em 3 passos: i) clonar o repositório e entrar nele; ii) configurar usuário (nome e email); e depois de fazer os programas: iii) adicionar, confirmar e enviar! 
                    git clone https://gitlab.com/hvescovi/prog23.git
                cd prog23

                git config user.name "Hylson"
                git config user.email "hvescovi@gmail.com"

                ... abra o VSCode no gerenciador de arquivos
                ... altere seus programas
                ... crie códigos
                ... faça as atividades
                ... e pra finalizar, atualize o repositório com os 3 comandos abaixo!

                git add *
                git commit -m 'mensagem'
                git push
